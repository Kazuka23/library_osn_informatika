## 📖 Deskripsi Umum

> **Library Competitive Programming** adalah repository yang dirancang untuk menyimpan koleksi solusi permasalahan dari berbagai platform *competitive programming*, khususnya **TLX (TOKI Learning Center)**. Repository ini merupakan dokumentasi perjalanan belajar dalam dunia pemrograman kompetitif menggunakan bahasa **C++**.

Repository ini dapat digunakan sebagai:
- 📚 **Referensi belajar** bagi siapa saja yang baru memulai *competitive programming*
- 🏅 **Persiapan OSN Informatika** dengan kumpulan solusi dan teknik algoritma
- 🔍 **Sumber belajar mandiri** untuk memahami struktur data dan algoritma melalui contoh nyata
- 💡 **Inspirasi pendekatan** dalam memecahkan soal-soal pemrograman yang kompleks

Saat ini fokus utama repository berada pada soal-soal dari **TLX TOKI**, dan ke depannya akan diperluas ke platform lain seperti **Codeforces** dan **AtCoder**.

## 🛠️ Cara Pakai

### 1️⃣ Clone Repository

Pastikan kamu sudah menginstall **Git** di komputermu. Jika belum, download di [git-scm.com](https://git-scm.com).

```bash
# Clone repository ini ke komputermu
git clone https://github.com/Kazuka23/library_competitive_programming.git

# Masuk ke folder repository
cd library_competitive_programming
```

---

### 2️⃣ Install C++ Compiler

Kamu memerlukan compiler C++ untuk menjalankan kode di repository ini.

<details>
<summary><b>🪟 Windows</b></summary>

**Opsi A — MinGW-w64 (Rekomendasi untuk pemula):**
1. Download installer dari [winlibs.com](https://winlibs.com) atau [mingw-w64.org](https://www.mingw-w64.org)
2. Jalankan installer dan ikuti langkah instalasinya
3. Tambahkan path `C:\mingw64\bin` ke **Environment Variables > PATH**
4. Verifikasi instalasi:
   ```bash
   g++ --version
   ```

**Opsi B — MSYS2:**
1. Download dari [msys2.org](https://www.msys2.org)
2. Buka terminal MSYS2 dan jalankan:
   ```bash
   pacman -S mingw-w64-x86_64-gcc
   ```

</details>

<details>
<summary><b>🍎 macOS</b></summary>

```bash
# Install Xcode Command Line Tools (sudah termasuk g++)
xcode-select --install

# Atau install via Homebrew
brew install gcc

# Verifikasi
g++ --version
```

</details>

<details>
<summary><b>🐧 Linux (Ubuntu/Debian)</b></summary>

```bash
# Update package list
sudo apt update

# Install g++ compiler
sudo apt install g++ -y

# Verifikasi
g++ --version
```

</details>

---

### 3️⃣ Pilih IDE / Code Editor

Berikut adalah rekomendasi IDE yang umum digunakan dalam *competitive programming*:

| IDE / Editor | Keterangan | Download |
|---|---|---|
| **VS Code** ⭐ | Ringan, banyak extension, cocok untuk pemula | [code.visualstudio.com](https://code.visualstudio.com) |
| **CLion** | IDE profesional khusus C/C++ (berbayar, gratis untuk pelajar) | [jetbrains.com](https://www.jetbrains.com/clion/) |
| **Code::Blocks** | IDE gratis, ringan, sudah include compiler | [codeblocks.org](https://www.codeblocks.org) |
| **Dev-C++** | Sederhana, cocok untuk kompetisi | [sourceforge.net](https://sourceforge.net/projects/orwelldevcpp/) |

> 💡 **Tips:** Untuk competitive programming, **VS Code** + ekstensi **C/C++** dari Microsoft adalah kombinasi yang sangat direkomendasikan!

---

### 4️⃣ Compile & Jalankan Kode

```bash
# Compile file C++
g++ -o output nama_file.cpp

# Jalankan program
./output          # Linux / macOS
output.exe        # Windows

# Compile dengan optimasi (opsional, direkomendasikan untuk CP)
g++ -O2 -o output nama_file.cpp
```

---

Setiap file `.cpp` di repository ini mewakili satu solusi dari satu soal. Kamu bisa:
